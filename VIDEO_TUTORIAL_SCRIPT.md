**[Screen: Repository page]**

"Now let's upload the ToraShaout files.

3. In GitHub, click 'Add file' ‚Üí 'Upload files'

## üìπ Scene 5: Launch Codespaces (1.5 minutes)

"Perfect! Now for the magic - launching Codespaces.

1. Click the green 'Code' button

3. Click 'Create codespace on main'

- Creating your development environment
- Installing Node.js
- Installing all dependencies
- Setting up VS Code extensions

4. Type 'npm run dev' and press Enter

[Wait for server to start]

5. Click 'Open in Browser'

[Browser opens with ToraShaout]

And there it is! The complete ToraShaout landing page - running in the cloud, accessible from any device!"
